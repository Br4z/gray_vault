The following list includes the modified bindings; the default bindings are not listed here.

# Fortnite controls

## Battle royale

- Use: `y`.

### Movement

- Crouch: `ALT`.

- Auto run: `z`.

### Weapon

- Pickaxe: `MB4`.

- Slot 1: `1`.

- Slot 2: `TAB`.

- Slot 3: `2`.

- Slot 4: `3`.

- Slot 5: `CTRL`.

### Building

- Wall: `q`.

- Floor: `c`.

- Stairs: `f`.

- Roof: `MB5`.

- Trap: `NUMPAD 5`.

- Edit: `e`.

### Communication

- Place marker: `b`.

### Misc

- Emote: `[`.

- Toggle map: `x`.

- Cursor mode: `v`.

## Save the world

### Weapon

- Consumible slot: `CTRL`

- Gadget slot 1: `4`.

- Gadget slot 2: `5`.

- Use: `y`.

### Building

- trap picker: `NUMPAD 5`.

- repair / upgrade: `t`.
