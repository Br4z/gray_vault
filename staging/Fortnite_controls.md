The following list includes the modified bindings; the default bindings are not listed here.

# Fortnite controls

## Movement

- Crouch: `ALT`.

- Sprint: `SHIFT`.

- Auto run: `z`.

- Left: `a`.

- Backward: `r`.

- Right: `s`.

## Weapon

- Reload: `TAB`.

- Pickaxe: `MB4`.

- Slot 1: `1`.

- Slot 2: `MOUSE WHEEL UP`.

- Slot 3: `2`.

- Slot 4: `3`.

- Slot 5: `CTRL`.

- Use: `p`.

## Building

- Wall: `q`.

- Floor: `c`.

- Stairs: `t`.

- Roof: `MB5`.

- Trap: `4`.

- Edit: `f`.

## Communication

- Place marker: `v`.

## Misc

- Toggle map: `x`.

- Cursor mode: `d`.
