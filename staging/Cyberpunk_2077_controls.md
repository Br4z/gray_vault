# Cyberpunk 2077 controls

The following list includes the modified bindings; the default bindings are not listed here.

## Movement

- Left: `a`.

- Backward: `r`.

- Right: `s`.

- Crouch (toggle): `ALT`.

## Weapons

- Cycle weapon: `c`.

- Next weapon: `MOUSE WHEEL UP`.

- Previous weapon: `MOUSE WHEEL DOWN`.

- Select first weapon: `1`.

- Select second weapon: `2`.

- Select third weapon: `3`.

- Fist or melee: `4`.

- Quick attack: `q`.

- Reload: `TAB`.

- Use combat gadget `MMB`.

## UI

- Cyberware systems: `f`.

- Open phone: `b`.

- Skip dialogue: `j`

- Open photo mode: `k`.

## Vehicle

- Call vehicle: `d`.

- Turn left: `a`

- Brake: `r`.

- Turn right: `s`.

- Exit vehicle: `t`.

- Cycle lights: `d`.

- Radio `TAB`

- Draw / withdraw weapon: `v`.

## Braindance

- Fast-forward: `f`.

- Rewind: `q`.

- Analysis mode: `p`.

- Restart: `TAB`.

## Other

- Scan (hold): `p`.
