# Quality dimensions

## Accuracy

Data correctly represents the real-world value it should represent.

## Completeness

All required data is present.

## Consistency

Data is the same across systems and does not contradict itself.

## Timeliness

Data is up to date and available when needed.

## Validity

Data follows the correct format, rules and constraints.

## Uniqueness

No unwanted duplicates exist.
