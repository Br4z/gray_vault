# Static assets

They are files that are served to clients without being generated again for each request. The content stays the same until the file itself is changed. Common examples include HTML files, CSS stylesheets, JavaScript files, images, fonts, icons and videos.

These assets are usually used to build the visible and interactive parts of a website or application. Because they do not need server-side computation each time, they can often be delivered efficiently and cached easily.
